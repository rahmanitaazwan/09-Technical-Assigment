1. Apa itu CSS Framework?
CSS Framework sendiri merupakan kumpulan kode CSS yang dapat digunakan untuk mengembangkan website.
CSS Framework adalah hasil formula codingan website yang telah siap digunakan. 
Sehingga programmer tidak perlu menghabiskan banyak waktu dan tenaga untuk membuat formula CSS website baru dari awal.
